# 旅游景点推荐系统 python480

## 项目概述

旅游景点推荐系统是一款基于Python和Django框架开发的应用，旨在为用户提供景点信息查询、旅游咨询、行程分享等功能的综合性平台。

## 技术栈

- **后端技术**：Python、Django框架
- **前端技术**：VUE
- **数据库**：MySQL
- **开发工具**：PyCharm

## 功能模块

### 前台管理

- 景点信息
- 旅游咨询
- 行程分享
- 个人中心
- 我的收藏

### 后台管理

- 景点信息管理
- 景点分类管理
- 酒店信息管理
- 行程分享管理
- 系统管理

## 系统角色

- 普通用户：查看景点信息，分享行程，收藏喜欢的景点等
- 管理员：管理景点信息，分类，酒店信息以及系统设置

## 运行环境

- Python环境
- Django服务器
- MySQL数据库
- VUE前端框架

## 部署步骤

1. 安装Python环境
2. 安装Django框架
3. 安装MySQL数据库
4. 配置数据库连接
5. 部署前端VUE代码
6. 运行Django应用

## 目录结构

```
project/
│
├── frontend/           # 前端VUE代码目录
│
└── backend/            # 后端Django代码目录
    ├── apps/            # Django应用目录
    ├── static/          # 静态文件目录
    ├── templates/       # 模板文件目录
    └── manage.py        # Django项目管理文件
```
